Detailed Description

Architecture of the Expander pipeline stage.

The standard shift registers chain is splitted into a number of chains equals to the unrolling factor

According to the SHA-256 specification, the 16 initial words which the input message is splitted into are in big-endian order, causing a reverse sorting of the input message, which must be taken in little-endian order. This makes it necessary to reverse the input of the expander, and this reversal must in turn be taken into account when splitting the expander into stages. Namely, the right shift becomes a left shift when reversed
