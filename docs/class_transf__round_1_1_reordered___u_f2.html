Detailed Description

Spatially-reordered, 2-unrolled architecture of the transformation round block.

Implementation of the architecture originally proposed in H. E. Michail, G. S. Athanasiou, V. Kelefouras, G. Theodoridis, and C. E. Goutis,"On the Exploitation of a High-Throughput SHA-256 FPGA Design for HMAC", ACM Trans. Reconfigurable Technol. Syst., vol. 5, no. 1, p. 2:1–2:28, 2012.
